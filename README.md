# Smart-Road-Health-Monitoring-System
An AI-powered end-to-end web application that automates the detection of road surface defects and provides standardized engineering metrics for municipal triage.

## 🚀 Features
- **AI-Powered Diagnostics:** Uses YOLO11 to detect Potholes, Alligator Cracks, Transverse Cracks, and Longitudinal Cracks.
- **Automated Civil Logic:** Computes road health index scores (0-100) adhering to the standard ASTM D6433 Pavement Condition Index (PCI).
- **Dual-Portal Workflow:** Features a Citizen Reporting Portal and an authenticated Municipal Operations Dashboard.
- **Automated Bureaucracy:** Instantly generates downloadable, official PDF inspection reports using FPDF.

## 🛠️ Tech Stack
- **Frontend/Backend:** Streamlit (Python)
- **AI Core:** Ultralytics YOLO11 & PyTorch
- **Database:** SQLite3
- **Report Generation:** FPDF
